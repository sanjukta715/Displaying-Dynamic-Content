# Dynamic Blog (PHP + MySQL)

## Setup
1. Import `posts.sql` and `users.sql` into your MySQL database.
2. Update DB credentials in `db.php`.
3. Run the project using localhost (XAMPP/WAMP).
4. Visit http://localhost/your-project/index.php

## Features
- Fetches blog posts dynamically from MySQL.
- Shows author name using JOIN.
- Displays creation timestamp in readable format.
